# Polyline (opens new window)

名称 类型 是否必填 默认值 描述
show Boolean 可选 true 如果显示此折线，设为 true，否则设为 false。
width Number 可选 1.0 这显得宽度(以像素为单位)
loop Boolean 可选 false 是否在第一个和最后一个折线间添加线段成为闭环线段
material Material (opens new window) 可选 Material.ColorType 材料
positions Array.<Cartesian3 (opens new window)> 可选 位置
id Object 可选 唯一标识符
distanceDisplayCondition DistanceDisplayCondition (opens new window) 可选 指定在距相机多远处显示此折线的条件
const redLine = viewer.entities.add({
  name: "RedLine",
  polyline: {
    positions: Cesium.Cartesian3.fromDegreesArray([-75, 35, -125, 35]),
    width: 5,
    material: Cesium.Color.RED,
    clampToGround: true,
  },
});

const greenRhumbLine = viewer.entities.add({
  name: "GreenLine",
  polyline: {
    positions: Cesium.Cartesian3.fromDegreesArray([-75, 35, -125, 35]),
    width: 5,
    arcType: Cesium.ArcType.RHUMB,
    material: Cesium.Color.GREEN,
  },
});

viewer.zoomTo(viewer.entities);

# PolylineGraphics (opens new window)

名称 类型 是否必填 默认值 描述
show Property (opens new window) | boolean <可选> true 指定折线可见性的布尔属性。
positions Property (opens new window) | Cartesian3 (opens new window) <可选> 它指定定义线带的 Cartesian3 位置的数组。
width Property (opens new window) | number <可选> 1.0 以像素为单位指定宽度的数字属性。
granularity Property (opens new window) | number <可选> Cesium.Math.RADIANS_PER_DEGREE 如果 arcType 不是 ArcType.NONE，则指定每个纬度和经度之间的角距离的数字属性。
material MaterialProperty (opens new window) | Color (opens new window) <可选> Color.WHITE 指定用于绘制折线的材料。
depthFailMaterial MaterialProperty (opens new window) | Color (opens new window) <可选> 用于指定折线低于地形时用于绘制折线的材料。
arcType Property (opens new window) | ArcType (opens new window) <可选> ArcType.GEODESIC 折线段必须遵循的线类型。
clampToGround Property (opens new window) | boolean <可选> false 指定是否应将折线固定在地面上。
shadows Property (opens new window) | ShadowMode (opens new window) <可选> ShadowMode.DISABLED 指定折线是投射还是接收来自光源的阴影。
distanceDisplayCondition Property (opens new window) | DistanceDisplayCondition (opens new window) <可选> 指定将在距相机多远的地方显示此折线。
classificationType Property (opens new window) | ClassificationType (opens new window) <可选> ClassificationType.BOTH 指定此折线在地面上时是否对地形、3D 瓷砖或两者进行分类。
zIndex Property (opens new window) | number <可选> 0 指定用于排序地面几何图形的 zIndex 的属性。仅当 clampToGround 为 true 并且支持地形上的折线时才有效。

# PolylineGlowMaterialProperty (opens new window)

名称 类型 是否必填 默认值 描述
color Property (opens new window) | Color (opens new window) 可选 Color.WHITE 用于线的 Color 属性。
glowPower Property (opens new window) | Number 可选 0.25 指定发光强度，作为总线宽的百分比
taperPower Property (opens new window) | Number 可选 1.0 指定锥形效果的强度，以总线条长度的百分比表示。如果 1.0 或更高，则不使用锥形效果。
const glowingLine = viewer.entities.add({
  name: "GlowingLine",
  polyline: {
    positions: Cesium.Cartesian3.fromDegreesArray([-75, 37, -125, 37]),
    width: 10,
    material: new Cesium.PolylineGlowMaterialProperty({
      glowPower: 0.2,
      taperPower: 0.5,
      color: Cesium.Color.CORNFLOWERBLUE,
    }),
  },
});

# PolylineOutlineMaterialProperty (opens new window)

名称 类型 是否必填 默认值 描述
color Property (opens new window) | Color (opens new window) 可选 Color.WHITE 用于线的 Color 属性。
outlineColor Property (opens new window) | Color (opens new window) 可选 1.0 指定轮廓的 color 属性
outlineWidth Property (opens new window) | Number 可选 1.0 指定轮廓的宽度（以像素为单位）
const orangeOutlined = viewer.entities.add({
  name: "OrangeLine",
  polyline: {
    positions: Cesium.Cartesian3.fromDegreesArrayHeights([
      -75,
      39,
      250000,
      -125,
      39,
      250000,
    ]),
    width: 5,
    material: new Cesium.PolylineOutlineMaterialProperty({
      color: Cesium.Color.ORANGE,
      outlineWidth: 2,
      outlineColor: Cesium.Color.BLACK,
    }),
  },
});

# PolylineDashMaterialProperty (opens new window)

名称 类型 是否必填 默认值 描述
color Property (opens new window) | Color (opens new window) 可选 Color.WHITE 用于线的 Color 属性。
gapColor Property (opens new window) | Color (opens new window) 可选 1.0 指定间隙的颜色
